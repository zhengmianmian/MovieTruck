# MovieTruck
An online cinema system that enables you to view movie playlists and reserve your seat.

## Tech Stack
- SQL Server
- ASP.Net 8.0
- React
- [Zod](https://zod.dev/)
- [Tanstack](https://tanstack.com/query/latest/docs/framework/react/overview)
- [React hook form](https://react-hook-form.com/)

## Run the app
Go to Server folder:
```
dotnet run
```
